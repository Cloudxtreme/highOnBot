High-On-Bot


Terminal over Facebook Messenger, running continuously as a background process.



About

It is an attempt to provide command line utility "remotely" via facebook messenger bot so that the user can work on his/her machine from anywhere.

Installation

In very simple steps you can get this functionality to use the command line utility remotely on all the three platforms namely MacOS, Windows and Linux.

How it works?

We use the SSH protocol and hence using the credentials of main machine on our chatbot we are able to work on the main machine remotely.
